### :rocket: Projets

#### 🖥️ [HOMELAB](#) — *mai 2023 - aujourd'hui*
Infrastructure personnelle en constante évolution, terrain d'expérimentation pour l'infra et le DevOps.
- Virtualisation avec **Proxmox VE** sur un vieux PC recyclé
- VMs dédiées : serveur web (Apache, MySQL/MariaDB, PHP), serveur média (Jellyfin + Sonarr + Bazarr + Seanime), **OPNsense** (firewall, WAN failover, DHCP, DNS)
- Supervision du parc avec **Homepage / Grafana**, Pi-hole (adblocking)
- Accès distant : RDP, TightVNC, redirection de port sur box FAI, VPN **Tailscale** (accès sans passer par la box)
- NAS virtuel (**Rockstor**) pour partage NFS
- **Docker Compose** pour orchestrer la stack média sur une seule VM
